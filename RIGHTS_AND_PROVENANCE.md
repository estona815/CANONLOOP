# Rights and Provenance

All visible campaign media depicts an entirely fictional masked electronic performer. The images
were generated specifically for CANONLOOP through the built-in OpenAI image-generation tool. The
project owner authorized their use in the public demo and hackathon submission. No uploaded user
media, web-search image, real artist, label, album cover, or company logo is present.

Each file is identified by SHA-256 in `ASSET_MANIFEST.json`. Replay B2 paths describe the intended
object layout and are marked unverified until a real authenticated storage round trip succeeds.
