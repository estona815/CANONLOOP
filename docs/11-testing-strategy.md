# Testing Strategy

The backend suite validates contracts, thresholds, all eight agents, traversal and image guards,
safe packaging, fake-client B2 round-trip logic, API disclosure, and real Genblaze no-key execution.
The web suite validates shared schemas, release counts, parent linkage, lint, strict TypeScript, and
production build. Browser QA covers desktop, mobile, interactions, package download, and CSP/errors.
