# Design System

Accepted concepts: `artifacts/design/landing-concept.png` and `workspace-concept.png`.
Tokens: deep ink `#04101a`, surface `#071824`, cyan `#27d8f4`, amber `#f3a516`, green
`#42d27f`, red `#ff5160`, warm-white text, cool-gray dividers, 10–12px radii. Typography is a
modern system grotesk; icons use a consistent 1.5–2px Lucide outline. The container model is open
bands on landing and rail/canvas/inspector in the workspace, not a generic card grid.
